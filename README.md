# AuthFirestorePOC

This repository contains the proof-of-concept project which demonstrates the usage of the Firebase Authentication and Cloud Firestore for Android
  
## Requirements  
- Android Studio >= 3.2.1  
- Gradle >= 4.7   
- Kotlin >= 1.3.20
